# Support

## Getting Help

For usage questions and implementation support:

- Open a GitHub Discussion in this repository
- Open a GitHub Issue with clear reproduction details

## What to Include

- Node.js version
- Package manager and lockfile in use (`npm`/`pnpm`)
- Relevant logs (without secrets)
- Steps to reproduce
- Expected vs actual behavior

## Security Issues

For security issues, follow [`SECURITY.md`](./SECURITY.md) and do not disclose publicly.

## Response Expectations

This is a community-maintained project. Response time varies by contributor availability.
